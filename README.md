#SQL Bascis

- SQL: Structured Query Language, managing and manipulating relational databases.
- Database: An organized collection of structured information, or data
- Table: The basic unit of data storage in a relational database. It consists of rows and columns
- Row (Record/Tuple): Represents a single, complete entry of data in the table
- Primary Key: A column that uniquely identifies each row in a table
- Foreign Key: A column in one table that links to the primary key of another table. This is how relationships between tables are established
- For Main Categories of SQL
  - DDL (Data Definition Language): Define database schema (CREATE, ALTER, DROP commands)
  - DML (Data Manipulation Language): INSERT, UPDATE, DELETE
  - DQL (Data Query Languge): Retrieve Data (SELECT)
  - TCL ( Transactional Control Language): Manage the changes made by DML statements (COMMIT, ROLLBACK)
- Key Clauses: WHERE, JOINs
- Aggregate Funcs: COUNT(), SUM(), MIN(),MAX(), GROUP BY
